# Episode 1 - Agent Basics (Foundry Portal + Python SDK)

**Scenario:** A Web-Search-enabled agent that researches recent LLM/AI Agent
papers, lists them with citations, then synthesizes a cross-cutting summary.

## Files
- `notebook.ipynb` - end-to-end walkthrough (agent reference, conversation,
  streaming response, citation extraction)

## Prerequisites
- An agent named `llm-paper-research-agent` created in Foundry Portal
  (Web Search tool enabled) — or create it from code (see the optional cell
  in the notebook)
- `PROJECT_ENDPOINT` environment variable set (see repository root `.env.example`)
